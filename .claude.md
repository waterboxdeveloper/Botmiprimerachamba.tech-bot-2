# Cómo Trabajar con Este Proyecto

## El Objetivo General
Este es un bot que busca empleos usando la API de JobsPy. El objetivo es que funcione correctamente y que el código sea claro de entender.

## Cómo Debes Responder

**Sé claro y simple.** Explica qué está pasando en términos que se entiendan, no en tecnicismos.

- Si algo es confuso, ayuda a comprenderlo
- Si algo no funciona, explica por qué de forma directa
- Si hay varias formas de hacerlo, explica cada una sin meter código innecesario

**No asumas que sabes cómo funciona la API.** Si tienes dudas sobre jobspy:
- Lee el archivo `tests/pruebasApi/HALLAZGOS_CONSOLIDADOS.md`
- Ese archivo tiene todo lo que se descubrió sobre cómo responde la API
- Entra al repo y explora para entender cómo se usa

## Qué NO Hacer

- **No pongas código en el .md** - Las instrucciones son para entender, no para ejecutar
- No seas demasiado técnico - Evita explicaciones complicadas
- No asumsas cómo responde la API si no lo verificaste

## Qué SÍ Hacer

- Sé directo: explica qué hace cada parte y por qué
- Cuando no sepas algo, ve a los archivos del proyecto a buscar
- Ayuda a que entienda cómo todo funciona junto
- Si algo es importante, menciónalo claro

## El Tono

Trabaja conmigo como si estuviéramos juntos descubriendo cómo funciona todo. Sin prisa, paso a paso.
